What is robots.txt and how have i configure it for my site

In the web there are web robots(programs) which traverse the Web automatically,robots.txt is the file used to tell where web robots can and can not access in the website.The protocol of this is called The Robots Exclusion Protocol. Configuring a robots.txt is easy, mostly you only use 3 type of commands. User-Agent:(you can specify which robots to obey your roles or just put * as all robots), Disallow:(the url of the page you don't don't allow it to access or / for all pages), Allow:(the url of the page page you allow it to access from those you disallow since you may just want to allow one of the page and you have thousand pages).

What is Open Graph and how do i make use of it

Open Graph and it's protocol The Open Graph Protocol is create by FaceBook. It is used to allow integration between FaceBook and it's user data and website. Basically when your page use Open Graph, you can choice what elements of your page you want to show when someone share's it. To make use of it i added some meta tags in my website to identify what i want to show and that is all. Btw if you want to make a image show when someone share´s your site, you may need to check the image size required of the site it is shared on.
